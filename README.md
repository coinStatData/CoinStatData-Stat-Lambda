# CoinStatData-Python-Stat-Lambda

Window:
- py -3 -m venv venv
- venv\Scripts\activate
- python3 -m pip install chalice

Mac:
- python3 -m venv venv
- venv/bin/activate


```
## Deploying

$ pip freeze > requirements.txt
$ cat requirements.txt
$ chalice deploy
```

```
## Local setup

$ pip install -r /path/to/requirements.txt
$ flask --app app --debug run
```
